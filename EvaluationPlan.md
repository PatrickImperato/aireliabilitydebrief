# Evaluation plan

## Offline checks

Schema validation pass rate
Reference validity pass rate
Template compliance rate

## Output quality checks

Objective coverage
How many objectives appear in the debrief

Claim precision
How many claims cite at least one valid source

## Acceptance gates

Gate 1
Output validates
All claims have valid sources

Gate 2
Objective coverage meets target on labeled set
Invalid claim rate stays below target

Gate 3
Human review workflow
Track edit rate and edit reasons
